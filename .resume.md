## 핵심 기술 사항 (반드시 숙지)

### BAT 파일 인코딩 문제

- `dev-one-click-setting-kit.bat`는 **CP949(EUC-KR)** 인코딩
- Edit 도구로 직접 수정 불가 — 한국어 부분이 깨져서 매칭 실패
- **반드시 Python 패치 스크립트로 수정해야 함**

### Python 패치 기본 구조

```python
# -*- coding: utf-8 -*-
import shutil

BAT_PATH = r"D:\AI_Dev_Work\2026y\26y_04m_22d_dev-one-click-setting-kit\dev-one-click-setting-kit.bat"

# 1. 백업 먼저
shutil.copy(BAT_PATH, BAT_PATH + '.bak')

# 2. CP949로 읽기
with open(BAT_PATH, 'r', encoding='cp949') as f:
    content = f.read()

# 3. 수정 (old → new)
content = content.replace(OLD_TEXT, NEW_TEXT)

# 4. CP949로 다시 쓰기
with open(BAT_PATH, 'w', encoding='cp949') as f:
    f.write(content)
```

### 구현 전략

모든 13개 수정사항을 **단일 Python 스크립트** (`upgrade_v1.1.0.py`)로 한 번에 적용.
이유: CP949 파일을 여러 번 열고 닫으면 줄 번호 변화 위험 → 한 번에 처리.
